Features

This package stops themes from installing Snowboard, this is very useful if a theme has Anemone as a dependency but you want to use your engine of choice.

Known issues

  • This package will seriously break Snowboard! It's also not needed as it's meant to fix an issue with Anemone and not with Snowboard.
  • This package may conflict with iThemer, I reached out to DODGraphics about this!
